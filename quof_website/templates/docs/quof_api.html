Quof Api

To interact with the API of our site, you must register on it. After registering on the main page, go to the Api tab. This page will display your api key.

Download Api

To download, just send a request to this link:

127.0.0.1:5000/api/<api_key>/download

If you need to download a specific version, add it after the slash.

127.0.0.1:5000/api/<api_key>/download/<version>

For now, that's all! In the future, the API will expand significantly!
